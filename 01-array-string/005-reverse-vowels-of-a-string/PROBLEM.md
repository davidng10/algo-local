# 345. Reverse Vowels of a String
**Difficulty:** Easy
**Link:** https://leetcode.com/problems/reverse-vowels-of-a-string

## Description

Given a string `s`, reverse only all the vowels in the string and return it.

The vowels are `'a'`, `'e'`, `'i'`, `'o'`, and `'u'`, and they can appear in both lower and upper case, more than once.

## Examples

**Example 1:**
```
Input: s = "IcesCreAm"
Output: "AcesCreIm"
```

**Example 2:**
```
Input: s = "leetcode"
Output: "leotcede"
```

## Constraints

- `1 <= s.length <= 3 * 10^5`
- `s` consists of printable ASCII characters.
