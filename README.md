# Algo 75

## Description

Purely just to practise on the plane, not intended to replace or replicate leetcode.

## How to use

```bash
npm install
npm test          # watch mode
npm run test:run  # single run
```

Each problem lives in its own folder with:

- `PROBLEM.md` — problem description
- `ANSWER.md` — reference solution + explanation
- `solution.ts` — implement your solution here
- `test.ts` — vitest tests
